URL Basics

The Uniform Resource Locator is another word for web address. The URL can be made of words or an Internet Protocol address.
A web address, URL, follows these syntax rules: scheme://prefix.domain:port/path/filename

Scheme

Type of Internet service, most common are http/https.

Commmon Schemes

HyperText Transfer Protocol - Used for unencypted common websites.
Secure HyperText Transfer Protocol - Used for encrypted secure websites.
File Transfer Protocol - Used for downloading/uploading files.
File - Used for files on your computer.

Prefix

Domain prefix, the default for http is www.

Domain

The Internet domain name, example being w3schools.com.

Port

The port number at the host, 80 is the defualt for http.

Path

Specifies a path of the server, if this isnt included then it goes to the root folder.

Filename

Specifies the file.
