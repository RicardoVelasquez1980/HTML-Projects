Inline

This is what you have been doing so far to implement CSS.
It is the style = attribute.
It is usually used to apply a unique styling to an HTMl element.

Internal

This is where you decide the styling within the HTML but rather than adding it in each element you
write it in the head element.
This styling will apply for ALL HTML elements unless you overwrite them using the inline method.

External

This involves a seperate style.css file which decides the styling for the entire site.
This file is called within the HTML head element as follows:
< link rel = "stylesheet" href = "style.css" >
