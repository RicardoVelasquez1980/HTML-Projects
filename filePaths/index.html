Absolute File Path

An Absolute File Path is the full URL to a file. The URL can link to an external file, from another site, or internally, from the same site.

This image is an Absolute File Path because it uses a full URL to link to an external file.

Relative File Path

A Relative File Path points to a file on the current page. It is typically best to use these when possible.

File In Same Folder As Current Page
file.type
File In A Folder In The Current Folder
folder/file.type
File In A Folder At Root Of Current Page
/folder/file.type
File In A Folder One Lever Above From Current Folder
../file.type
