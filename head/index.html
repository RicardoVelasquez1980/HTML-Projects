Intro

The Head is an element which contains metadata. Metadata usually defines the document title, character set, styles, script, and more. It also isnt displayed.

Title

The title is the title of the document and is shown in the browsers title bar, the name on the pages tab. The title is required since it is used for SEO and to allow search engine algorithms to decide the order when listing pages in search result.

Style

Just the styling for the single HTML document.

Link

The link element defines the relationship between the current document and an external resource.
Link is mainly used to reference external style sheets or add an icon next to the page title.

Meta

The meta element is usually used to specify the character set, page description, keywords, author of the document, and viewport settings.
Meta isnt displayed but it is used by the browsers to know how to display content or reload the page, by search engines to get the keywords, and by other web services.

Scripts

Scripts can be in the head element but it is best to put them just before closing the body. This way when loading a script it wont delay loadin the HTML.
Also if the script were to not load then the everything under it wont load.

Base

The base element specifies the base URL for all relative URLs in the page.
So if you were to do images/face.png, it will become baseURL/images/face.png
There can also only be one base element in the entire document. Look at the code to see how to make a base URL.
