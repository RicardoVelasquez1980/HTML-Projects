Class Basics

Classes are made in the styling sheet and can then be an attribute to ANY element.

Classes can be reused on as many elements whether the same element or not.

More On Classes

Multiple classes can also be on a single element. In order to add more classes to an element just separate the class names by a space.

The styling is still the same here even though the ordering of the class names are different. This is because it is the ordering of the Class
within the styling sheet that decides which is applied first.
