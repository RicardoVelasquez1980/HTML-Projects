Entity Basics

As you have noticed, literally typing < or > within a paragraph, or anything else which contains text, has its issues within the HTML itself.
These characters are reserved characters. And so this is where entities come in. Entities are used in order to display reserved characters. Below is a list of these reserved characters.
Look at the code for how its done

  •    
  • < <
  • > >
  • & &
  • " "
  • ' '
  • ¢ ¢
  • £ £
  • ¥ ¥
  • € €
  • © ©
  • ® ®
