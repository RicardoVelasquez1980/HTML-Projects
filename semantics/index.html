Semantic Meaning

As brefily mention in computercode, semantics are elements which provide meaning.
The following are semantics which are used to define parts of a site.

Sections

Sections are used to, well, define a section. A section typically contains a group of content, usually with a heading.
Examples of this are the following:

  • Chapters
  • Intros
  • Contact Info

Article

The Article Element is used to speficy self contained content such as a forum post, blog post, comments, product cards, etc.

Header

What is a Header?

A Header typically contains heading(s), icons/logos, authorship info, and more.
Although you can have multiple header elements in a site, headers cant be nested in a footer, address, or another header element.

Footer

The footer typically contains authorship info, copyright info, contact info, sitemap, back to top links, and similar sites.

Author: Ricardo Velasquez

Nav

Although I dont show an example of this, nav is used when creating a navigation bar, in most cases.
A Nav Bar is a list of links which take the user to another related site or to a bookmarked section in the site itself.
